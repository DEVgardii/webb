MLB Total World Series Wins by Franchise

Select a bar to see the number of losses for the team.

This is a fairly simple column chart with some bars representing each MLB team with the height showing the amount of World Series they have won. If you move your mouseover or select a bar the corresponding label on the map should change to red.

Some things I learned making this map is that there are a large proportion of teams in the northeast and that the Yankees winning all the time made the MLB boring while I was growing up. It seems like some non historically dominant teams have been making it deep into the playoffs lately though. Hopefully the league keeps it up.

The information for this map came from a wikipedia page about world series champs, with Baseball Team location provided by a github repo maintained by James Fee and the map data was taken from Mike Bostocks Atlas repo with some basic datasets mostly derived from US census bureau derived shapefiles.
